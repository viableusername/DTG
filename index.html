D->G V1.1.2

MAX and MIN take a message ID as input
you need the ids of the oldest message and the newest message in the range of messages you want to save
MAX: newest message
MIN: oldest message
once these 2 parameters are set you can input your DHT.txt file (browse button)
there is currently a problem with the download process so you'll need to do the following:

  1. input your file or press the process button if you've already added your file but nothing happened
  2. click the text below MAX/MIN that's begging to be clicked
  3. open the "DHT-to-gsheets.csv" that was just downloaded
  4. ctrl+a (select all)
  5. ctrl+v (paste)
  6. save the changes and you can now import the file to gsheets
