D->G V1.2.0

MAX and MIN take a message ID as input
you need the ids of the oldest message and the newest message in the range of messages you want to save
MAX: newest message
MIN: oldest message
once these 2 parameters are set you can input your DHT.txt file, you'll get a CSV in your downloads shortly after uploading your file

Normal Mode

the default mode, use this if you're not trying to find wrong counts or messages that match a regular expression

!WRONG Mode

this will create a CSV of all incorrect counts within the selected range (only works for counts 100,000 to 999,999)

REGEX Mode

input regex in the regex and process file to get messages that match your regex
